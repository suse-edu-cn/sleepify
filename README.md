# 睡了么

![当前版本](https://img.shields.io/github/package-json/v/suse-edu-cn/sleepify)
![开源协议](https://img.shields.io/github/license/suse-edu-cn/sleepify)

你说得对，但是睡了么是一款用于对抗亲爱的何导的校园睡积分比赛的 Web 程序，基于 Next.js 框架。本应用 UI 组件库为 [MDUI v2](https://www.mdui.org/zh-cn/)，API 设计主要用作中转和定时任务（你懂的），图标是号主的自设。~~由于号主是个懒鬼，~~ 本应用的 UI 以外的多数逻辑处理由 AI 完成，希望各位包含orz

在此之外还制作了面向 Android 设备的 Compose 版本，现已推送到 [compose/main](https://github.com/suse-edu-cn/sleepify/tree/compose/main) 分支，欢迎到 Releases 里下载。感谢 [@HuangZhuoRui](https://github.com/HuangZhuoRui) 的帮助。

为保护隐私，此处不公开提供原程序样本和上游 API 文档。
